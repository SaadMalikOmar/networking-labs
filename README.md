# Networking Labs — Design, Routing & Configuration

Hands-on network design, configuration, and testing from my university networking module (CI5210), built and verified in **Cisco Packet Tracer**. Both pieces scored the **highest mark in the cohort**.

Networking fundamentals underpin security work: you can't monitor, segment, or defend a network you don't understand. These labs cover addressing, routing, and multi-site design end to end.
